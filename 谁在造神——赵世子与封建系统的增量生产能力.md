# 谁在造神？——赵世子与封建系统的增量生产能力

## *一个关于"封建系统在吃饱之后获得的不是加速器，是一条生产线"的个人观察*

---

如果你在2022年之前问一个福建长乐人"赵世子是谁"，对方大概率会愣一下。

如果你在2024年春节之后问，对方的回答可能是"宅男、爱漂亮、衣服不好看不肯出门"——像一个追星粉丝在介绍自家爱豆的人设。

赵世子这个"神"，2023年才正式面世。设计周期约两年。设计师是个00后，叫林滨。设计灵感来源于国产仙侠动画《眷思量》里的一个角色。神像的面部是"动漫建模脸"——因为本来就是按动漫脸画的。

他没有经书。没有谱系。没有神学。没有功德事迹。没有妈祖那种"海上救人"的功绩KPI。没有关公那种"忠义"的道德叙事。

"他是谁的儿子"这一问题的答案也是一笔烂账：有人说是五福大帝赵公明之子，但五福大帝的原始传说里压根没有"世子"这回事。台湾宗教数据库搜不到。当地老人想不起来游神里有个叫赵世子的。在学术考据层面——他不存在。

但这不妨碍他**作为神被制造、被崇拜、被运营、被消费、被写入非遗文创盲盒。**

赵世子不是孤例。他属于一个叫"世子天团"的产品矩阵——张大世子（爱抽华子、宠弟狂魔）、张二世子（贪玩爱出门）、华光大世子（三只眼、曾设定为8岁爱吃QQ糖、后"成长"为帅哥吸女粉）。每个都有独立人设、独立粉丝群、独立周边产品线。

我想搞清楚一件事：**一个不存在的神是怎么被造出来的。** 这件事的意义不在于它发生在福建——在于它暴露了封建操作系统在物质丰裕条件下的**增量生产能力。** 上一篇我写了"吃饱让封建从低功耗升级到性能模式"，讲的是存量——彩礼、面子、续香火这些老机制被加速了。但存量升级不是上限，上限是**系统能造新东西。** 赵世子就是这个生产线的展品。

### 一个必须前置的辨析：这不是"抖音造网红"

抖音每天造出成百上千个"男神""女神"——算法推送、直播打赏、MCN签约、三个月过气换下一批。那条流水线的产品是**流量节点**：可替换、可遗忘、没有神圣性、不进庙。

赵世子这条流水线的产品是**神明**：有托梦授权的、有庙的、被非遗盖章的、有东家作为私人所有者的。他不是被算法选中的——他是被设计师画出来、被托梦合法化、被游神仪式"激活"的。

两条流水线表面像——都有直播、都有打赏、都有粉丝。但产品进了不同的仓库。抖音网红进的是MCN合同，赵世子进的是庙。前者三五年后没人记得，后者一旦进了游神队伍、被非遗认了证、被做成了盲盒——他就从"可以被遗忘的流量"变成了"被官方记录的文化遗产"。

本文讨论的是后一条流水线。

## 一、造神五步法

造神五步法不是一个业务流程——它是格式化学底层协议（名→分→礼→行→实）在增量生产模式下的全速实例化。每一步对应传动轴的一个环节：

- **设计稿 = 名。** 系统先创造一个名字——不是神迹积累了几百年后"被命名"，是设计师画出来的那一刻就有了名。
- **托梦授权 = 分。** 名需要空间位置。托梦不是赋予神圣性——是在神谱里给这个名字分配一个位置。"赵世子是赵公明之子"——不管真假，分一旦成立，后续环节就激活了。
- **上线运营 = 礼。** 分有了位置，行为规范就长出来了。怎么拜、穿什么衣服、能不能出门、跟粉丝怎么互动——这些都是礼。不是东家"发明"的，是粉丝在分的位置上一拥而上生成的。
- **饭圈化运营 = 行。** 礼内化之后，粉丝自动执行。不需要命令——"打榜"就是新式的献资，"应援"就是新式的游神。你自己会做。
- **合法性认证 = 实。** 最终产出——非遗证书、国企盲盒、文旅GDP贡献值。一个被格式定义过的神。不是"信不信"的问题——它已经进了博物馆和盲盒。那就是实。

五步不是阴谋，是传动。没有人设计了整条链条——但每个环节的参与者都在自己的位置上完成了自己被分配的那一段。

### 第一步：设计稿

传统的民间造神通常需要几百年。一个地方上发生了一件无法解释的事——一个人被雷劈死但尸体不腐、一个渔夫在风暴中平安回来——民间开始讲述，慢慢形成"灵验"的口头传统，有人捐钱修小庙，香火积累到一定程度被道士和尚收编进正规神谱，或者被官府认定为"淫祀"打掉。这个过程是自发的、缓慢的、没有设计师的。

赵世子的路径不一样。

他有一个设计师。林滨，长乐本地人，克凤村林氏族人。他花了约两年时间完成了赵世子的面部设计——按照动漫审美标准，不是按照任何宗教图像志传统。设计成果是一个"建模脸"——在抖音和小红书的评论区里，他被反复形容为"霸总脸""好帅""想嫁"。

没有任何人跪在庙里求了几百年灵验之后才决定长什么样。**神的长相是一个产品设计决策。**

### 第二步：托梦授权

设计稿有了，需要神学执照。

"神明托梦指示要塑一尊新像。"这句话本身不是什么新发明——托梦、扶乩、降神，是中国民间信仰里最古老的神谕机制。关公显灵、妈祖托梦、各路地方神的"灵验"传说，几百年来的剧本大同小异：神迹发生（某人绝处逢生）→口耳相传几十年→托梦确认"是我保佑的"→修庙塑像。托梦一直是授权流程的最后一步，在民间叙事发酵足够充分之后才登场——它是**收尾**，不是**开场**。

赵世子的区别不在"用了托梦"。在**托梦被压缩到了两年设计周期里，并且从收尾变成了开场。**

传统造神：神迹→口传几十年→托梦确认→修庙塑像。赵世子：设计稿→托梦→直播上线。同一年完成。以前是"神迹在前、托梦在后"——先有事再有名。赵世子反过来——**先有设计稿后有托梦。** 名在实之前。神像还没雕好，人设已经定了、直播间已经开了、粉丝已经在等"哥哥今年出不出来"了。

"托梦"在这里的功能不是"确认一个已经存在的神迹"。是**为一件尚未完成的产品提供神学合法性。** 工具是旧的，新的是速度和方向——以前是神迹拉动托梦，现在是托梦拉动生产线。

### 第三步：上线运营

2023年春节，赵世子首次加入长乐游神队伍。

游神本身是一个已有约四百年历史的传统——把庙里的神像抬出来巡境，驱邪祈福。但赵世子加入的不是传统游神。他加入的是一个已经被短视频平台重新编码的**内容产品。**

游神过程全程直播，实时在线观看人数可达十几万。主播引导观众刷礼物。神像有固定机位。粉丝在弹幕里刷"世子好帅""世子今天穿什么"。

### 第四步：饭圈化运营

赵世子的运营团队被称为"东家"——这个词本身就很说明问题。不是"庙祝"、不是"理事会"、不是"信众代表"。是"东家"——所有者的意思。

赵世子的东家网名"林老六"，人**在美国。** 他的姐姐在国内负责神像的日常维护。东家决定神像"出不出门""接不接受外村邀请"。东家在直播间里说："我的心情就是赵世子的心情。"

这不是比喻。这是**所有权声明。**

东家为赵世子建立了完整的人设："宅男属性，很难请出门，注重外表，必着黑金龙袍。"粉丝据此产生了情感投射——不是"信仰"，是"粉"。"哥哥今天好帅""东家对哥哥不好——造型是旧款、头饰掉了""榜一大姐付费摸世子脸"。

在饭圈的认知架构里，赵世子不是一个应该跪拜的神圣存在。他是一个需要被**守护**的、有性格缺陷的、可以投射情感的商品。拜神和追星使用同一套行为模式——打榜（献资）、应援（游神出巡）、周边（盲盒扇子）、站姐（直播机位）。

### 第五步：合法性认证

2024年底，长乐区产业投资发展集团——一家国企——联合文创公司推出了《游神天团》IP盲盒。包含七爷、八爷、胜金少娘、赵世子等角色及隐藏款华光大帝。在三坊七巷、烟台山等福州热门景点销售。

这个动作是整条造神流水线的**认证环节。** 国企出品=官方盖章。盲盒=赵世子从"信仰"被重新定义为"文化消费品"。一旦进了文旅商品的筐，所有关于"淫祀""造神""借神敛财"的质疑都可以用"文创创新"四个字挡回去。

非遗+文创+国企=三把保护伞同时撑开。合法性闭环完成。

## 二、东家：神明的新阶级

传统庙宇的权力结构大致是：**神明 → 庙祝/道士/僧侣（神职人员）→ 信众（供养者）。** 神职人员在神和人之间充当中介，理论上只服务神明、不拥有神明。

东家创造了一个全新的角色：**神明 → 东家（所有者/经纪人）→ 粉丝（消费者）。**

东家不是神职人员。东家不做宗教仪式（或者外包给别人做）。东家的核心工作是**运营**：设计人设、管理直播、控制IP版权、决定巡游排期、处理粉丝关系、开发周边产品。东家本质上是一个**偶像经纪公司**，神像是旗下艺人，粉丝是消费者兼自发推广者。

这个模型比传统宗教高效得多：

- **不需要神学院。** 不需要读经、不需要修行、不需要被任何教团认证。你是东家因为你说你是东家。
- **不需要信仰。** 粉丝不需要"信"赵世子——他们只需要觉得他帅、萌、惨、值得守护。信仰是沉重的、需要承诺的；粉是轻量的、可以随时换的。
- **收入结构多元化。** 传统的香火钱靠的是信众自愿投放。东家的收入来自：直播打赏、周边销售、巡游受邀费用（东家承认"受邀出门会收费"）、神像租赁、献资名单（公开张贴的单村单场活动可达43万元）。
- **离岸可控。** 东家在美国。神像在福建。IP在云端。粉丝在全球。没有任何一个国家的监管体系覆盖这个配置。

## 三、钱从哪来，怎么花

（本节内容曾考虑独立成篇，现精简后作为本文的经济基础分析。完整的跨境资本流通分析不在此展开。）

福建游神活动的资金规模在公开报道的碎片中大致可见：

- 长乐区潭头镇厚福村2024年正月初十和正月二十六两日花费：**破百万。**
- 潭头镇某村单次游神献资名单：单笔最高3万元，最低2000元，合计**近43万元。**
- 有"东家"团队全年多场活动总花费达**八位数（千万级）。**
- 神像"塔骨"贵的一尊可达数万元至小几十万，纯金配饰另算。
- 资金来源："80后出钱，90后管理，00后出力。"出钱最多的是父辈——包括大量海外华侨。

**80后出钱。** 这代人的财富积累从哪来？部分来自国内生意，但长乐的特殊性在于——**20多万长乐人生活在美国。** 2024年正月初十游神就有200多名厚福人从美国返乡参加。

这些人给游神捐的钱，是美元还是人民币？汇款路径是什么？有发票吗？需要缴税吗？谁审计这笔资金？

如果一个人在美国赚了钱，通过正规银行渠道汇回中国，需要申报、有额度限制、大额可能触发反洗钱审查。如果他把一笔钱捐给福建某个村的游神活动——以现金形式放进"献资"的筐里——没有任何人会问。

不是有人在**设计**这个机制。是机制自己**长出来了。** 当宗族网络的跨境延伸（华侨）叠加上一个完全现金化、匿名化、不被审计的献资传统——它就自动构成了一个不受现代金融监管体系覆盖的跨境资本流通管道。

没有人需要策划"洗钱"。但管道就在那里，敞开着。

## 四、淫祀2.0：为什么这次的造神不一样

历史上"淫祀"一词用来打击未经官方认证的民间祭祀。孔子说"非其所祭而祭之，名曰淫祀，淫祀无福。"历朝历代取缔淫祀的逻辑大致相同：正统宗教有经典、有谱系、有国家册封、有道德教化功能。淫祀什么都没有——只有一个据说很灵的东西。

但赵世子的情况让"淫祀"这个词显得不太够用。

淫祀1.0的特征是：自发产生、口耳相传、缓慢积累、地方性、官方打压。淫祀2.0（赵世子模式）的特征是：**设计驱动、一夜上线、全网传播、饭圈化运营、官方盖章（国企盲盒、非遗文创）。**

区别在于：

| | 淫祀1.0 | 淫祀2.0 |
|---|---------|----------|
| 诞生速度 | 几十年到几百年 | 两年 |
| 生产方式 | 自发涌现 | 设计师+团队+托梦 |
| 传播介质 | 口耳相传 | 短视频+直播 |
| 信徒结构 | 本地信众 | 全网粉丝 |
| 收入模式 | 香火钱 | 打赏+周边+盲盒+受邀费 |
| 与官方关系 | 被打击 | 被认证（非遗文创） |
| 所有权 | 集体共有 | 东家私有 |

最关键的变化在最后两行。

淫祀1.0不是任何人的私有财产。淫祀2.0的赵世子是**有明确所有人的**——东家林老六。神像不是公共信仰设施，神像是**私人IP。** 国企非遗盲盒付没付IP授权费？付给谁？付了多少？这些钱从文创公司到长乐产投到东家再到美国——整条资金链上没有任何一笔在阳光下可追溯。

**一个私人持有的、可在国际间移动的、享有免税地位的、披着非遗保护外衣的无形资产。** 这已经不是在讨论"封建迷信"了。这是在讨论一个全新的资产类别。

## 五、回到框架：封建系统的增量生产能力

在上一篇《吃饱大封建运动》里，我写了：当人类不再需要为吃饱消耗全部精力时，剩余的全部资源流入了宗法系统准备了两千年的管道——彩礼、面子、续香火。那是**存量激活。** 旧机制加速运行。

但赵世子演示了更深一层：系统不只是把旧代码跑得更快。它在**写新代码。**

封建操作系统在物质丰裕条件下的增量生产能力包括：

1. **造新神。** 不需要经典、不需要神迹、不需要几百年积累。设计师+托梦+短视频就够了。
2. **造新阶级。** 东家——神明的私人所有者兼经纪人。游走于信仰、文创、直播三个领域的跨界资本。
3. **造新金融管道。** 游神献资+海外华侨+匿名现金+非遗免税=一个事实上的跨境资本通路。
4. **造新合法性。** 官方不镇压淫祀——官方给淫祀发非遗证书然后卖盲盒。合法性不是"被允许存在"，是"被纳入地方文旅GDP"。

最让人不安的结论是：**这些不是阴谋，是涌现。** 没有人坐在一个房间里决定"我们要利用封建系统建立跨境资本网络"。是系统在物质条件充足后，自我组织出了这些结构。林滨画赵世子的时候可能真的只是想"创新传统文化"。东家人在美国可能真的只是"没法回国所以远程管理"。献资名单上的人可能真的只是"想给村里出份力"。

但他们合在一起做的事，任何一个人都没有完整意图做的事——就是制造了一个世界级的、无人监管的、披着神圣外衣的金融-文化-流量复合体。

系统不需要阴谋家。系统只需要足够多的人各自执行他们被分配的那一小段脚本。拼起来就是一台完整的机器。

## 附记：盲盒里的神明

写到这里，想起长乐产投出的那款《游神天团》盲盒。

一个消费者花几十块钱拆一个盲盒，可能拆出赵世子、可能拆出华光大世子、可能拆出隐藏款华光大帝。拆盒的瞬间跟拜神的瞬间是同一个心理结构——你在等待一个**高于你的存在**降临到你手里。区别在于：拜神的跪在蒲团上，拆盲盒的坐在直播间里。前者投香火钱，后者刷礼物。前者求平安，后者求隐藏款。

神像从庙里搬到了盲盒里。但搬运过程中发生了一件事：**神像在庙里的时候，它属于所有人。进了盲盒，它属于东家。**

这是我这三篇论文想说的全部事情的一个微型演示。
