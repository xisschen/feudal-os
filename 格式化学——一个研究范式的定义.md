# 格式化学：一个研究范式的定义

## *关于"什么是格式化学、研究什么、怎么研究"的元理论框架*

---

这篇文章不是案例分析。是工具箱。

前面三篇论文——狗哨方法论、吃饱大封建、瞬时造神——各自处理了一个经验现象。但它们共享同一套底层逻辑。这篇文章的任务是把那套底层逻辑抽出来，命名、定义、划定边界、给出操作手册。

它的功能是让后来者——无论研究中国、印度、日本还是中东——知道"格式化学"这四个字是什么意思，以及"我能不能用它的框架来分析我的案子"。

---

## 一、定义

**格式化学（Formatics）** 研究的是：**一套前现代的权力-伦理复合体，如何在现代化的物质条件下，从低功耗生存模式切换到全功率运行模式，并在此过程中产生增量——新规则、新阶级、新金融管道、新合法性。** 研究对象是该复合体用来**格式化人类行为的编码机制**：人类如何被塑造成不需要外部强制、就能自动执行系统指令的单元。

"格式化"这个词来自计算机：磁盘被格式化之后，写入的文件系统决定了数据怎么存、怎么读、谁能写。对人做格式化，意思是一套规则被写入到认知底层——不是"我知道这件事该这么做"的层面，而是"这件事本身就该这么做"的层面。彩礼不是"我选择给"，是"反正得给"。续香火不是"我觉得该生儿子"，是"不生儿子我这辈子没过"。

格式化学的核心研究问题：

1. **格式化是怎么完成的？** 有哪些机制在往人的认知底层写入规则？这些机制需要什么物质条件？
2. **格式化在不同物质条件下怎么切换模式？** 饥饿时为什么降频？吃饱后为什么全速？不同经济水平的同一套规则输出是什么？
3. **格式化系统为什么能产生增量？** 不只是重复执行旧规则——为什么物质充足之后它开始造新规则、新阶级、新神、新金融工具？
4. **格式化系统的退出成本是什么？** 为什么有人明明被系统磨损，仍然留在系统里加速运行？没有备用剧本是什么意思？

## 二、核心概念

以下概念构成格式化学的基本词汇表。它们是分析工具，不是描述性标签。

### 2.1 封建OS

一个文明在农业时代发育出的、以**血缘差序**为路由协议、以**礼/名分/避讳/道统**为编码机制、以**自我执行**为强制手段的权力-伦理复合体。

关键特征：

- **不是暴力维持的。** 强制手段是次要的、补充性的。主要手段是被格式化者的自我管理。
- **不是顶层设计。** 没有"总设计师"。系统是通过无数人各自执行局部规则而涌现出的整体秩序。
- **跨地域变体共享一个底层架构。** 中国的宗法系统、印度的种姓系统、日本的家制度、欧洲的封建等级——差别在具体规则，相同的在格式化逻辑。

### 2.2 格式化（Formatting）

将社会规范写入个体认知底层、使其以**默认协议**方式运行的过程。完成格式化的人：

- 不再意识到"这是一套人为规则"——规则变成"自然如此"
- 执行规则不需要计算得失——"反正就得这样"
- 对违反规则的人产生自发排斥——不需要上级命令

格式化的完成标志是**自我命令**。拿枪逼着你给彩礼——强制。你自己觉得不给彩礼不对劲——格式化。

### 2.3 低功耗模式 / 性能模式

格式化系统在两种物质条件下的运行状态：

| | 低功耗模式 | 性能模式 |
|---|----------|----------|
| 触发条件 | 生存资源紧张 | 生存资源充裕 |
| 规则执行 | 简化、延迟、打折扣 | 全量、实时、无打折 |
| 面子竞争 | 地板效应（差距小） | 无上限（差距可以无限拉大） |
| 退出可能 | 物理上无法退出 | 物理上可以退出，认知上不一定 |
| 增量生产 | 没有——维持现状 | 有——新规则、新阶级、新工具 |
| 系统开销 | 绝大多数能量用于生存 | 剩余能量全部流入格式化管道 |

低功耗不是"没有封建观念"。是**规则都在，但CPU没空跑**。性能模式不是"封建回来"。是**规则一直在，CPU终于可以跑了**。

这个模型的推论：**现代化不是封建的敌人，是封建等待了三千年的给养。** 不是"现代化的物质成果应该消解封建但没有消解"——是"封建一直在等待足够的物质条件来全功率运行"。

### 2.4 增量生产（Incremental Production）

格式化系统在性能模式下不仅加速执行旧规则，而且**产生旧规则集合中没有的新东西**。包括：

- **新神。** 不是民间自发的、缓慢积累的灵验传说——是设计师画出来、托梦合法化、直播上线的神圣产品。
- **新阶级。** 不是传统的神职人员（庙祝、道士、僧侣）——是神明的私人所有者兼经纪人（东家）。
- **新金融管道。** 不是有意设计的洗钱网络——是宗族跨境网络+献资匿名现金传统+非遗免税地位自动涌现出的无监管资本通道。
- **新合法性。** 不是被官方镇压的淫祀——是被官方非遗认证、走进国企盲盒、纳入地方文旅GDP的神明。

增量生产是格式化学区别于传统社会学"封建残余"叙事的关键。"残余"意味着存量在消退——只是还没消完。增量意味着**系统在制造新东西**——不是残余，是生长。

### 2.5 狗哨机制（四形态）

狗哨（dog whistle）：发送者传递了一个表面无害的信号，接收者——且**只有被格式化的**接收者——能解码出其中的指令。这一机制有四种不可相互还原的形态：

1. **无主 / 礼。** 指令没有可追溯的来源。"谁规定彩礼20万？"——没人。但同时所有人都知道"就是这个价"。没有主语的社会强制。

2. **空洞 / 名分。** 信号的内容是空的——"赵世子是高颜值神明"——但接收者能用自己的格式化认知把它填满。一个词、一个名号、一个身份标签，不需要携带信息，只需要触发对方已有的编码。

3. **负空间 / 避讳。** 信号的力量在它没说出来的部分。"力工"替代"农民工"——缺的那一笔（"力"少出头变"刀"）比写出来的部分携带更多信息。被避讳的东西本身成为最响亮的在场。

4. **递归 / 道统。** 上一轮狗哨的结果成为下一轮的输入。避讳产生新词，新词产生新避讳——循环无限收敛到"道统"这个不可质疑的终点。

狗哨不是阴谋——没有人"设计"了这些信号。它是格式化系统自然运行时的信号传输协议。

## 三、研究对象与边界

### 3.1 格式化学研究什么

- 任何社会中被格式化到"默认协议"层级的行为规范——执行者不再意识到自己在执行规则
- 这些规范的编码、传输、执行机制
- 物质条件变化如何改变格式化系统的运行模式
- 格式化系统从"维持自身"到"增量生产"的切换条件

### 3.2 格式化学不研究什么

- **制度层面有明确书面法律并由国家暴力强制执行的规范**（如刑法、税法）。格式化学关心的是"没有人拿枪逼着但你照做了"的东西。
- **纯粹的个人偏好或心理现象**。不是所有无意识行为都是格式化——"我习惯喝咖啡"和"我习惯觉得没儿子就白活了"不是同一个东西。
- **有意设计的政治宣传或商业广告**。这些有明确作者、有可追溯的来源、有可测量的效果。——狗哨无主，宣传有主。这是根本区别。

### 3.3 边界案例：什么时候"有主"变成"无主"

最重要的边界案例发生在**最初有意的规则逐渐失去其设计者、被社会自然化为"就是这样的"**的时刻。

一个长官当年下令"这个村不许提那件事"。两代人后，没有人记得那次命令。但没有人提那件事。这不是宣传——宣传的源头已经消失了。剩下的就是格式化。

格式化学的专长正是在这里：在"权力"和"默认秩序"的模糊边界上识别出格式化的运行痕迹。

## 四、方法论

### 4.1 概念推演

格式化学依赖理想型（ideal type）构造。从经验现象中提取核心结构、暂时剥离经验杂音、构造一个逻辑一致的纯概念模型。它不是实证研究——它的验证不在统计显著性，在**结构映射的精度**和**案例的典型性**。

概念推演的风险是脱离经验。防御策略：**每个概念必须有至少两个经验锚。** 提"格式化"，必须同时给出被格式化行为和被非格式化对照行为的实例。提"增量生产"，必须在纸上放一个具体的增量产物（赵世子的设计稿、某村游神的献资名单），不允许纯抽象跳跃。

### 4.2 历史比较

格式化系统的模式切换（低功耗→性能）需要一个时间轴上的对照。关键方法是：**同一个行为，在饥饿时代和饱食时代的执行方式、强度、成本之间的系统比较。**

- 同地不同时：同一地区彩礼金额的长期面板数据
- 同行为不同执行度："续香火"在溺女流行区和当代二胎追儿子的比较
- 异文明同构：中国彩礼通胀 vs 印度嫁妆通胀的平行比较

比较的陷阱：**不要拿一个文明的理想型去验证另一个文明的理想型。** 正确做法是从经验现象出发，再识别背后的格式化结构。

### 4.3 田野实证

概念推演和历史比较提供框架，田野实证填充血肉。格式化学需要多种类型的经验材料：

- **深度访谈。** 不是"你信不信"，是"你觉得不给彩礼行不行"——以测试格式化深度
- **参与观察。** 游神现场的资金流动可以观察到多少？献资名单公开张贴说明了什么？
- **档案挖掘。** 侨批书信中关于"寄钱回乡修庙"的表述。族谱中的财产转移记录。
- **数字踪迹。** 东家的直播数据、粉丝的弹幕内容、献资金额的时间序列。

田野的产出不是"证明"框架——是**把框架钉到经验材料上**，让后续研究者能定位到同一块经验土壤。

### 4.4 同行验证流程

1. **框架公开。** 核心概念、研究边界、方法论在公开平台上完整陈述。
2. **案例对齐。** 其他研究者用同一框架分析不同案例（不同地区、不同文明）。
3. **边界压力测试。** 框架在被应用过程中遇到无法解释的案例——框架修正或边界收紧。
4. **概念收敛。** 经过多轮修正后，核心概念的含义在学术社区中稳定下来。

## 五、与既有学科的关系

格式化学不取代任何既有学科。它坐在几条界线上：

| 既有领域 | 格式化学取其 | 格式化学不取 |
|---------|------------|------------|
| 社会学（费孝通传统） | 差序格局、礼治秩序 | 功能主义假设（系统都在维持均衡） |
| 人类学（Bourdieu） | 惯习、象征资本 | 对"策略性"的强调——格式化的行为不是策略，是默认 |
| 传播学 | 编码/解码、媒介效果 | "发送者→接收者"这个模型——狗哨无发送者 |
| 经济人类学（Goody） | 婚姻交换的历史比较 | 纯经济理性框架 |
| 宗教史 | 淫祀、造神、民间信仰的文献 | 教义分析和宗教体验的真实性判断 |

格式化学的核心贡献：把"权力如何让人自己管自己"这个问题从这几个学科的边缘提到中心，并为之建立一个独立的分析框架。

## 六、跨文明比较坐标

以下不是跨文明分析。是**为后来者留下的比较坐标**——指出其他文明中可能存在功能等价物，但不替他们完成分析。

### 6.1 格式化的跨文明变体

| 文明 | 格式化核心机制 | 可能的对应概念 |
|------|-------------|-------------|
| 印度 | 种姓×业报轮回 | 达摩（dharma）作为默认行为编码、嫁妆通胀对应彩礼通胀 |
| 日本 | 家制度（ie）× 义理-人情 | "义理"作为格式化语言、"世間体"（面子）的全社会监控 |
| 中东 | 部落世系×荣誉伦理 | 血亲复仇作为自我执行、聘礼（mahr）的格式化和非格式化差异 |
| 拉丁美洲 | 大庄园制×天主教等级 | 恩主-附庸作为人格化格式、救赎经济对应面子经济 |
| 撒哈拉以南非洲 | 祖先崇拜×长者政治 | 彩礼（lobola）的商业化和城市化效应 |

### 6.2 比较方法论

- **先挖透一个文明。** 用完整的概念工具链（封建OS→低功耗/性能→增量生产→狗哨）分析一个文明的一个经验领域，得出可检验的命题。
- **再平行移植。** 把这些命题作为假说带入另一个文明——不是为了"证明框架普适"，是为了**检验命题在哪个层级崩溃**。崩溃点就是比较的发现。
- **不搞"本质化"。** 格式化学不声称"中国自古以来就是格式化的"——它声称"在具体的历史物质条件下，格式化机制被激活到了某个程度"。比较对象也用同样的动态框架处理。

## 七、研究议程

以下不是本文已完成的工作。是需要后来者填充的空白。

### 7.1 实证验证

- **彩礼通胀面板数据。** 多省数十年的村级彩礼金额追踪，控制变量（性别比、人均收入、外出务工比例），检验"系统指令"假说与"纯供需"假说的解释力差异。
- **力工群体系统抽样。** 问卷调查+深度访谈，测量格式化深度（彩礼支付意愿、面子竞争强度、传宗接代执念程度），与物质条件指标的关联。
- **游神献资数据分析。** 公开献资名单的统计（金额分布、捐赠人地理分布、捐赠人职业/年龄），追踪海外华侨资金的汇入路径。

### 7.2 历史纵深

- **清代以来彩礼金额的系统追踪。** 利用方志、族谱、婚书、民间契约重建一条跨越低功耗期和性能切换期的彩礼长时序数据。
- **淫祀与正统的博弈历史。** 历代官方打击淫祀的公文 vs 民间祭祀的实际存续——这套攻防是否在非遗文创时代被颠覆？

### 7.3 跨文明比较

- **中国彩礼通胀 × 印度嫁妆通胀。** 两个农业文明在现代化进程中的婚姻支付平行比较——是否共享相同的格式化逻辑？
- **侨批 × 哈瓦拉（hawala）。** 中国的宗族-游神献资跨境资金流和阿拉伯-南亚的哈瓦拉非正式汇款系统——是否功能等价？

### 7.4 当代延伸

- **算法格式化。** 短视频推荐算法的偏好函数是否正在成为格式化的**新编码器**？算法的"无主推荐"（没人知道为什么被推到，但它就是推了）是否构成狗哨的新形态？
- **彩礼贷的金融化。** 银行把"完成系统指令"做成信贷产品——是金融在服务封建，还是金融在**把封建转换成可证券化的资产**？

## 八、局限

本文——以及格式化学作为一个研究范式——承认以下根本局限：

1. **"格式化"无法被直接观察。** 它是从行为中推断出的认知状态。你能看到一个人给了彩礼，你看不到"他觉得反正得给"。格式化是一个**操作化上限很高**的概念——不是不能测，但测起来比"收入""教育年限"难得多。
2. **理想型抽象的代价。** "封建OS"是理论构造——现实中的农村社会是宗法、社会主义行政、市场消费逻辑的三重叠加。从叠加态中单独剥离格式化部分，本身就是一种简化。简化是为了分析的清晰，但清晰不等于完整。
3. **提出者偏差。** 本文由格式化学概念的提出者撰写——在定义自己的研究范式时，存在天然的自我确认倾向。这一偏差的纠正只能来自外部：后来者用本文的框架去分析案例，发现哪里走不通——那些地方就是本文的局限发生处。
